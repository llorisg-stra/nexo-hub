# Añadir Recurso

Registra una URL, documento o referencia externa al proyecto.

## Pasos

1. **Recopilar información del recurso**:
   - URL o ruta del documento
   - Tipo: documentación, API, tutorial, herramienta, referencia
   - Descripción breve
   - ¿Por qué es relevante?

2. **Leer/analizar el recurso** (si es una URL):
   - Extraer los puntos clave
   - Resumir lo más relevante para el proyecto

3. **Registrar** en CONTEXT.md o en una sección de recursos

4. **Confirmar** con el usuario

## Notas

- Verificar que la URL es accesible
- Si es documentación de API, considerar usar /api en su lugar
- Guardar un resumen, no solo el link
