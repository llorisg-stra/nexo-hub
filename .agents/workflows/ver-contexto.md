# Ver Contexto

Muestra el contenido de CONTEXT.md formateado de forma visual y fácil de leer.

## Pasos

1. **Leer CONTEXT.md** del proyecto actual

2. **Formatear y mostrar**:
   - Resumen ejecutivo con emojis
   - Tablas de infraestructura
   - Estado por fases con % de progreso
   - Decisiones técnicas recientes (últimas 5)
   - Próximos pasos priorizados
   - Stats rápidos (módulos, extensiones, páginas)

3. **Mostrar "salud" de la sesión**:
   - Última actualización (días desde)
   - Tareas urgentes pendientes
   - Warnings si hay inconsistencias

## Notas

- Este workflow es solo lectura, no modifica nada
- Si CONTEXT.md no existe, sugerir /contexto para crearlo
