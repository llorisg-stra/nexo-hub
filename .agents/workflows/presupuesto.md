# Presupuesto

Actualiza y revisa los costes y gastos del proyecto.

## Pasos

1. **Revisar costes actuales**:
   - VPS / Infraestructura (OVH)
   - APIs externas (OpenAI, Supabase, etc.)
   - Dominios y SSL
   - Herramientas de desarrollo

2. **Actualizar registro** de gastos:

   ```markdown
   | Concepto | Coste/mes | Proveedor | Notas |
   |----------|-----------|-----------|-------|
   | VPS Dev  | X€        | OVH       |       |
   | VPS Hub  | X€        | OVH       |       |
   | API Keys | X€        | OpenAI    |       |
   ```

3. **Calcular totales**:
   - Coste mensual total
   - Coste por cliente (si aplica)
   - Proyección a 3/6/12 meses

4. **Identificar optimizaciones** posibles

## Notas

- Actualizar cada vez que se añade un nuevo servicio
- Considerar costes de escalado por nuevo cliente
