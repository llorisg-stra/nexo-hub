# Decisión Técnica

Registra una decisión técnica importante en CONTEXT.md.

## Pasos

1. **Preguntar al usuario**:
   - ¿Cuál es la decisión?
   - ¿Qué alternativas se consideraron?
   - ¿Por qué se eligió esta opción?

2. **Formatear la decisión**:

   ```markdown
   ### YYYY-MM-DD — [Título de la decisión]
   
   - ✅ Decisión: [qué se decidió]
   - 🔄 Alternativas: [qué se descartó y por qué]
   - 💡 Razón: [justificación]
   ```

3. **Añadir a CONTEXT.md** en la sección "Decisiones Técnicas"

4. **Confirmar** con el usuario

## Notas

- Las decisiones nunca se borran, son historial
- Si afecta a la arquitectura, actualizar también las tablas relevantes
- Usar fecha actual automáticamente
