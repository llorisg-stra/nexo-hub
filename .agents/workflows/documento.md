# Añadir Documento

Incorpora un documento externo al contexto del proyecto.

## Pasos

1. **Identificar el documento**:
   - Ruta del archivo
   - Tipo (PDF, DOCX, imagen, Excel, etc.)
   - Propósito (referencia, especificación, diseño, contrato)

2. **Procesar el documento**:
   - Si es texto: extraer contenido relevante
   - Si es imagen: describir contenido visual
   - Si es PDF: extraer texto y resumir

3. **Registrar en el proyecto**:
   - Copiar a carpeta de documentos si aplica
   - Añadir referencia en CONTEXT.md
   - Crear resumen con puntos clave

4. **Confirmar** con el usuario

## Notas

- No duplicar documentos grandes innecesariamente
- Siempre crear un resumen legible del documento
- Respetar confidencialidad de documentos sensibles
